# KOSYAK — сайт-витрина

Простой статический сайт: главная страница = магазин (каталог), карточка товара, контакты.
Работает на GitHub Pages, ничего не стоит, ничего не сломается сам по себе.

---

## Структура файлов

```
index.html              — главная страница, она же каталог (магазин)
product.html            — карточка товара (одна страница на все товары)
contact.html            — контакты
css/style.css           — весь дизайн (цвета, шрифты, отступы)
js/products.js          — ТОВАРЫ. Редактируешь только этот файл, чтобы менять каталог
js/main.js              — логика отображения (трогать не нужно)
images/products/        — фото товаров
```

---

## Как добавить новую футболку

1. Положи фото в `images/products/`, например `tee-04.jpg`
   (фото лучше заранее обрезать/сжать — квадратные или вертикальные кадры смотрятся лучше всего)
2. Открой `js/products.js`
3. Скопируй один блок целиком:

```js
  {
    id: "tee-04",
    name: "TEE 004",
    price: "25 €",
    image: "images/products/tee-04.jpg",
    description: "Короткое описание футболки.",
    meta: "Хлопок 100% · Прямой крой · Один размер"
  },
```

4. Вставь его в конец списка (перед закрывающей `];`)
5. Поменяй `id` (должен быть уникальным, без пробелов), `name`, `price`, `image`, `description`, `meta`
6. Сохрани файл — при следующей загрузке на GitHub сайт обновится сам, ничего больше делать не нужно

## Как убрать футболку

Удали её блок `{ ... }` целиком из `js/products.js`.

## Как поменять цену/название/описание

Открой `js/products.js`, впиши новый текст в кавычках у нужного товара, сохрани.

## Как редактировать текст на сайте (не товары)

- На главной теперь только товары (без заголовков и текста) — она в `index.html`, редактировать там нечего, кроме списка товаров в `js/products.js`
- Контакты (ссылки на Instagram/Telegram/почту) — файл `contact.html`, замени:
  - `https://instagram.com/ТВОЙ_АККАУНТ` на свою ссылку
  - `https://t.me/ТВОЙ_АККАУНТ` на свою ссылку
  - `mailto:почта@пример.com` на свою почту

Редактировать HTML-файлы можно прямо в GitHub: открой файл в репозитории → значок карандаша (Edit) → меняешь текст → Commit changes внизу страницы.

## Как опубликовать изменения

Раз сайт уже живёт в репозитории `kosyakentertainment.github.io`, дальше всё просто:

**Вариант А — через сайт GitHub (без установки чего-либо):**
1. Заходишь в репозиторий на github.com
2. Открываешь нужный файл или папку
3. Кнопка **Add file → Upload files** — чтобы добавить новые фото
4. Значок карандаша (Edit) на файле — чтобы поменять текст/код
5. Внизу страницы — **Commit changes** (это как "сохранить и опубликовать")
6. Через 30–60 секунд изменения появятся на `kosyakentertainment.github.io`

**Вариант Б — через git на компьютере** (если позже захочешь освоить):
```
git clone https://github.com/kosyakentertainment/kosyakentertainment.github.io.git
cd kosyakentertainment.github.io
# вносишь изменения
git add .
git commit -m "добавил новую футболку"
git push
```

## Про оплату (на будущее)

Сейчас на сайте нет оплаты — кнопка "Написать, чтобы купить" ведёт на страницу контактов.
Когда будешь готов(а) подключить оплату:

- **Stripe Payment Links** — создаёшь ссылку на оплату в личном кабинете Stripe под конкретный товар, вставляешь её вместо `contact.html` в файле `product.html` (заменить `href="contact.html"` на ссылку на оплату) или добавить в `js/products.js` отдельное поле `paymentLink` под каждый товар — скажи, помогу это подключить, когда дойдёт до дела.
- **Gumroad** — аналогично, тоже просто ссылка на кнопку.

Ничего менять на сервере/хостинге не нужно — просто ссылка на кнопке.

## Дизайн — если захочешь поменять

Все цвета и шрифты — в самом верху файла `css/style.css`, в блоке `:root`:

```css
--bg:    #ffffff;   /* фон */
--ink:   #111111;   /* основной текст/чёрный цвет */
--grey:  #8a8a8a;   /* серый текст (цены, подписи) */
```

Поменяй значения — и вся палитра сайта изменится сразу везде.
